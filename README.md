## alina-base

### Структура
- `src/`: вихідний код
- `src/outputs/`: результати/артефакти (генерації, експорти тощо)

### Швидкий старт
1. (Опційно) створіть venv

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Встановіть залежності

```bash
pip install -r requirements.txt
```
